# Company Policy Assistant Chatbot 🤖

An AI-powered chatbot that answers company policy-related queries using Retrieval-Augmented Generation (RAG). It leverages LangChain, FAISS vector search, and Hugging Face models to provide accurate and context-aware responses.

---

## 🚀 Features
- Answers company policy questions intelligently
- Uses FAISS for fast document retrieval
- Implements RAG pipeline for better accuracy
- Supports multiple policy documents (PDF/DOCX)
- Easy-to-use interface

---

## 🛠️ Tech Stack
- Python
- LangChain
- FAISS
- Hugging Face Transformers
- Streamlit / Flask (whichever you used)
